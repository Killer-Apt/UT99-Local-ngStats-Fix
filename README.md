# UT99 Local ngStats Fix

This is a community preservation patch for restoring local ngStats functionality in Unreal Tournament / UT99.

## Purpose

Older versions of Unreal Tournament allowed players to view local ngStats through the in-game stats menu. Over time, the original hosted files and support pages disappeared, making local ngStats difficult or impossible to use on modern installs.

This patch restores the needed local ngStats files so players can generate and view local match statistics again.

## Installation

1. Download the latest release ZIP from the Releases page.
2. Extract the ZIP into your Unreal Tournament installation folder.
3. Run:

   `NetGamesUSA\ngstats\ngStatsUT.exe`

4. View the generated local stats HTML page.

## Notes

The current version may still require manually launching `ngStatsUT.exe`. However, you may edit the UnrealTournament.ini and remove the .com from the NetGamesUSA.com path to restore in game menu functionality. (Alternatively, you may edit directory names in file to match the .ini)

## Tested With

- Unreal Tournament / UT99
- Steam/GOTY installs: Tested/Compatible
